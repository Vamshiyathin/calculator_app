Create a new repository on GitHub.
Clone the repository to your local machine using git clone.
Add your calculator application code to the repository by creating a new Python file and saving it in the root directory of the repository. 
Create a new Python file for your unit test code and save it in the root directory of the repository. 
Install any necessary dependencies for your application and tests. For example, if you are using the unittest module for testing, it should be included in Python's standard library.
